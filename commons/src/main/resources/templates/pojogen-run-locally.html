Plain Old Java Object Generator


How to run POJO generator locally

Download these three files:

pojogen.jar javaparser-core-3.3.2.jar MountainModel1.java



Open a Command Prompt and run: 
java -cp pojogen.jar;javaparser-core-3.3.2.jar com.marcosavard.library.javaparser.generate.SourceBasedPojoGeneratorMain MountainModel1.java

That will generate the .java files locally.
